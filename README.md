## Local development

Open the files directly in a browser or run a simple local server:

```powershell
cd "C:\Blank.app.webb"
npx http-server -p 5173 .
# or
py -m http.server 5173
```

Then open `http://localhost:5173/privacy.html` or `http://localhost:5173/terms.html`.
